# skills/

Vault backups of Claude skill definitions. Each subdirectory contains a `SKILL.md` file that defines a repeatable workflow.

**Live skills live in `~/.claude/skills/`.** The copies here are vault backups. If a skill is updated, the live version in `~/.claude/skills/` is the source of truth.
