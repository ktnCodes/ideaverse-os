# _personal/ -- Growth Domain

## What This Domain Covers

Career development, learning, and portfolio project notes.

## Folder Structure

```
_personal/
  CONTEXT.md        # You are here
  career/           # Resume, certifications, career strategy, eval workbooks
  learning/         # Study notes, courses, interview prep
  portfolio/        # Portfolio project notes and build logs
```

## Key Conventions

- Career files are dated when time-sensitive (e.g., `2026-04-resume-refresh.md`)
- Learning files are organized by topic, not by date
- Portfolio files reference the actual project workspace -- notes here, code elsewhere

## Routing

| Task | Go to |
|------|-------|
| Update resume or career plan | `career/` |
| Study or review learning material | `learning/` |
| Document portfolio project progress | `portfolio/` |
